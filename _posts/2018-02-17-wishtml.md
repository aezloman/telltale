## Html
---

### What is html?

 * Hyper Markup Language
 
 * The structure of websites
 
 * 1/3 of Web Design
 
### Why Html?

 * People use it for sites
 
 * It gives more freedom about the content
 
### Sample Html Code

```html
<html>
 <head>
  {{ Meta Tags , stylesheets and scripts }}
 </head>
 <body>
  {{Content}}
 </body>
</html>
```
